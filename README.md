## Setup

### Prerequisites

- Rust v1.33 or higher
- Fisher v3

### Getting Started

1. `cd` to the root of the **starship** repo
1. Install the **starship** binary:

    ```bash
    cargo install --path .
    ```

1. Install the fish shell prompt with fisher:

    ```bash
    fisher $PWD
    ```
